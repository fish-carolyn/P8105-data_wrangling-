## Options in *read\_*

The read function has some fun tools to help: - skip= X will remove the
first X rows and use the next as variable names - colnames = FALSE will
not to use the first row as varibale names, will be numbered columns
instead - na = c(NA, “19”, “.”) shows missing values are equal to NA or
“19” or “.”-\> replaces 19 with NA - coltypes= cols(Group= col_factor())

``` r
litters_df_edited= 
  read_csv("data/data_import_examples/FAS_litters.csv",
           na=c("NA", 19, "."), 
           col_types = cols(Group= col_factor(), `GD0 weight`=col_character())
           ) 
```
